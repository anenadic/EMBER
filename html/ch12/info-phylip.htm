PHYLIP

PHYLIP (the PHYLogeny Inference Package, © 1986-1995 by Joseph Felsenstein and the University of Washington) is a package of programs for inferring phylogenies (evolutionary trees) from multiply aligned sequences. Methods available in the package include parsimony, distance matrix, and likelihood methods, including bootstrapping and consensus trees. Trees can be displayed using PHYLIP's own DRAWGRAM or DRAWTREE programs, or may be generated at other sites (see PHYLODENDRON).

PHYLIP - Phylogeny estimation from distance matrix data

A protein sequence alignment is first used to compute a distance matrix, under different models of amino acid replacement. The distance between each pair of sequences is used to estimate the total branch length between the two. The models of amino acid substitution are:

  • The PAM (Point Accepted Mutation) matrices of Margaret Dayhoff.
  • The Jones-Taylor-Thornton model, a recalculation of the PAM matrices based on a much larger dataset.
  • The Kimura model (1983), an approximation of PAM distance by measuring the fraction of amino acids, p, that differs between two sequences. Though more rapid than using PAM or the Jones-Taylor-Thornton model, it does not take into account which amino acids differ or to what amino acids they change, so information is lost.
  • The Categories model, in which the amino acids are divided into groups, and transitions between them are based on the genetic code with greater resistance to of change between groups (categories).

Phylogenies are then estimated (from distance matrix data) using various models:

  • FITCH uses the "additive tree model", according to which distances are calculated from the sums of branch lengths between species. This approach does not assume an evolutionary clock.
  • KITSCH uses the "ultrametric" model, which is essentially the additive tree model with an evolutionary clock assumed.
  • NEIGHBOR uses both the Neighbor Joining Method and the UPGMA (average linkage clustering) method. Neighbor Joining is a distance matrix approach that produces an unrooted tree without the assumption of a clock, whereas UPGMA does assume a clock. The methods are very fast and can thus handle large data sets.

An alternative approach is to use PROTPARS, which estimates phylogenies from protein sequences using the parsimony method.
